# Optimization Methods (C#)

Проект реализует численные методы оптимизации функции двух переменных.  
Рассматриваются классические алгоритмы, широко используемые в прикладной математике, машинном обучении и инженерных задачах.

---

# Реализованные методы

В проекте реализованы три метода оптимизации:

- Градиентный спуск (Gradient Descent)
- Метод сопряжённых градиентов (Conjugate Gradient)
- Метод Ньютона (Newton Method)

---

# Структура проекта

## `Point2D`

Класс для работы с точками и векторами в двумерном пространстве.

### Поля:
- `X` — координата по оси X
- `Y` — координата по оси Y

### Методы:
- `Norma()` — вычисление длины вектора
- `Dot(Point2D)` — скалярное произведение

### Операторы:
- `+` — сложение векторов
- `-` — вычитание
- унарный `-` — смена знака
- `*` — умножение на число

## `Optimizer`

Основной класс, реализующий алгоритмы оптимизации.

### Конструктор:
```csharp
Optimizer(
    Func<Point2D, double> function,
    Func<Point2D, Point2D> gradient,
    Func<Point2D, Point2D>? newtonStep = null)

Параметры:
- function — функция для минимизации
- gradient — градиент функции
- newtonStep — функция шага метода Ньютона

# Подробно про методы
## Градиентный спуск (GradientDescent)
Движение в направлении антиградиента: x_{k+1} = x_k - α ∇f(x_k)

### Особенности реализации
- Используется backtracking (условие Армихо)
- Шаг автоматически уменьшается
- Устойчив к плохому выбору начального шага

### Критерии остановки
- Малая разница между точками
- Малая разница значений функции
- Малая норма градиента

## Метод сопряжённых градиентов (ConjugateGradient)
Использует улучшенные направления поиска: p_k = -∇f_k + β p_{k-1}

### Особенности
- Быстрее градиентного спуска
- Особенно эффективен для квадратичных функций
- Используется коэффициент β (формула Поляка–Рибьера)
- Периодически выполняется сброс направления

### Критерии остановки
- Малая разница между точками
- Малая разница значений функции
- Малая норма градиента

## Метод Ньютона (Newton)
Использует информацию о кривизне функции: x_{k+1} = x_k - H⁻¹ ∇f(x_k)

### Особенности
- Очень быстрая сходимость (квадратичная)
- Требует вычисления второго порядка производных
- Чувствителен к начальной точке

В реализации шаг Ньютона вычисляется через функцию MatlabFunc. Формулы получены заранее с помощью MATLAB.

### Критерии остановки
- Малая разница между точками
- Малая разница значений функции
- Малая норма градиента
